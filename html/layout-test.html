Responsive Breakpoints

Breakpoint Screen Size Layout Adjustments
Desktop > 1200px Full sidebar, all layouts as designed
Laptop 992px - 1200px Adjusted spacing, full functionality
Tablet 768px - 992px Sidebar stacks on top, modern/collage adjust to 1-2 columns
Mobile 576px - 768px All layouts single column, touch-optimized buttons
Small Mobile < 576px Compact UI, collage becomes single column
